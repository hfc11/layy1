初识HTML

HTML 文档应该有.html文件扩展名。通常一个站点的主页被命名为“index.html”。 “index”表示这是浏览器将加载的默认文件。命名 HTML 文件时使用小写字母。HTML 资源的名称通常用在 URL 中。 URL 通常区分大小写。

  • HTML 学习如何搭建页面结构和内容 (盖房子 毛坯房)
  • CSS 学习如何美化页面 (装修)
  • JavaScript 学习如何给页面添加动态效果
  • jQuery JS语言框架,简化原生JS开发
  • Bootstrap 前端框架对html/css/JavaScript/jQuery进行封装,目的是提高前端页面的开发效率
